# ATS Resume Builder | منشئ السيرة الذاتية

> أنشئ سيرة ذاتية احترافية متوافقة مع أنظمة ATS في دقائق

## ✨ المميزات

- 📄 **6 قوالب ATS** — كلاسيكي، احترافي، عصري، تنفيذي، بالصورة، عمودين
- 🔍 **فحص توافق ATS** — تقييم فوري مع نصائح تحسين
- 📥 **تصدير PDF** — تحميل مباشر بجودة عالية
- 🖨️ **عرض وطباعة** — فتح في متصفح جديد للطباعة
- 🌐 **عربي / English** — دعم كامل للغتين
- 📷 **رفع صورة** — للقوالب التي تدعم الصورة الشخصية

## 🚀 التشغيل

```bash
# تثبيت التبعيات
npm install

# إنشاء ملف .env
echo "GEMINI_API_KEYS=your_key_here" > .env
echo "PORT=3000" >> .env

# تشغيل السيرفر
npm start
```

ثم افتح: **http://localhost:3000**

## 📋 القوالب

| القالب | الوصف |
|--------|-------|
| كلاسيكي ⭐ | بسيط وأنيق — الأكثر توافقاً مع ATS |
| احترافي | عناوين بارزة بخلفية رمادية |
| عصري | خط أسود جانبي |
| تنفيذي | مينيمال راقي |
| بالصورة | صورة شخصية + كلاسيكي |
| عمودين | شريط جانبي + صورة |

## 🛠️ التقنيات

- **Backend:** Node.js, Express
- **PDF:** Puppeteer
- **AI:** Google Gemini (اختياري)
- **Frontend:** HTML, CSS, JavaScript
